Tap the character's right hand (yellow), then the left hand (cyan) on each frame. The renderer uses left-hand for mirrored facings (W / NW / SE) so the sword stays in the visual right hand of the mirrored character.
Right-hand data is pre-loaded from the existing anchors.json — by default you only need to tap left hands. Use the toggle button to redo a right hand if needed.
When done, Copy JSON and paste it to your assistant, or Download and commit it as public/sprites/player/anchors.json.
